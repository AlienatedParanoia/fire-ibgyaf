# F.I.R.E — Find. Involve. Reach. Engage.

> Your gateway to every opportunity.

**F.I.R.E** is a student-built platform that helps Singapore students (ages 13–18)
discover and keep track of extracurricular **competitions and clubs** — in one fast,
organised place, with reminders so the good ones never slip past.

Built with **Next.js (App Router)**, **Supabase**, and **Tailwind CSS**.
